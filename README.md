# Glanzz ✨ Frontend

## 简介

Glanzz 项目【前端-前台】

当前状态：【开发中(80%)...】

开源、前后端分离、分布式、微服务博客系统。

| ------------ | ------ | ------------- |

| vue-router   | 4.1.3  | vue-router    |
